#+TITLE: Transmute (Tool for Rapid Automated Normalization, Scaling, Metadata Updates, and Text Extraction)

#+AUTHOR: James Dyer
#+EMAIL: james@dyerdwelling.family

Transmute is a comprehensive media management and conversion suite for Emacs. It provides a native Elisp interface to powerful command-line tools like =ffmpeg=, =ImageMagick=, =exiftool=, =sox=, and =tesseract=.

Designed for Dired users, Transmute allows you to batch process images, videos, and audio files with a structured, keyboard-driven interface.

* Features

- *Images:* Convert, resize (crush/scale), upscale (GAN), rotate, brighten, auto-colour, crop, OCR (text extraction), convert to PDF, extract from PDF, montage creation, date-based organisation, WhatsApp filename fix, orientation reset, horizontal split, and Org capture gallery export.
- *Video:* Convert (H.264/AAC), shrink, rescale, rotate (R/L/CCW), reverse, speed up/slow down, trim (top/tail), cut, extract audio, remove/replace audio, background music injection (random library with fade-out), concatenate, double length, extract frames, from frames sequence, GIF conversion, and glitch/flip removal.
- *Audio:* Convert to MP3, normalise volume, and trim silence.
- *Metadata:* Interactive hierarchical tagging, renaming based on creation date/tags, EXIF management, and sidecar file (.xmp) synchronization.
- *Dired & Image-Dired Integration* -- Process marked files or the file at point in Dired buffers, and operate on the displayed image from the =*image-dired-display-image*= buffer. =C-c M= works in both thumbnail and display views.
- *Intelligent Filtering:* Commands automatically filter your selection to relevant files (e.g., Image commands only target images), allowing you to safely trigger batch operations on mixed-media folders.
- *Async Engine:* All operations run asynchronously in the background with automatic cleanup of temporary files.
- *Status:* Real-time progress tracking in the Emacs =header-line= and centralized logging in the =*transmute-log*= buffer.
- *Interface:* Compact dispatch-oriented *Transient* menu with nested sub-transients (Image, Video, Audio, Tags), plus a *Completing-Read* menu as an alternative.

* Dependencies

Transmute relies on the following tools being installed in your system path:

- =exiftool= (Metadata management)
- =magick= (ImageMagick 7+ for image processing)
- =ffmpeg= / =ffprobe= (Video and audio processing)
- =sox= (Audio normalisation)
- =tesseract= (OCR/Text extraction)
- =pdftoppm= (from =poppler-utils=, for PDF image extraction)
- =realesrgan-ncnn-vulkan= (Optional: for AI upscaling)
- =trash-put= (Optional: for safe file deletion)

* Installation

1. Clone this repository to your local machine:
   #+begin_src bash
   git clone https://github.com/captainflasmr/transmute.git ~/source/repos/transmute
   #+end_src
2. Add the following to your Emacs configuration:
   #+begin_src elisp
   (add-to-list 'load-path "~/source/repos/transmute")
   (require 'transmute)

   ;; Bind the main Transient menu
   (global-set-key (kbd "C-c >") #'transmute-menu)
   #+end_src

* Usage

In any Dired buffer or the =*image-dired-display-image*= buffer:
1. Mark the files you wish to process (in Dired/thumbnail view).
2. Press =C-c >= to open the Transmute menu.
3. Select the desired operation (e.g., =i c= for image convert).

When focused on the full-size display buffer (=*image-dired-display-image*=), the displayed file is automatically used as the target.

If no files are marked, Transmute will operate on the file at point.

** Menu Structure

The =transmute-menu= is a nested transient with a compact 2-row dispatch:

#+begin_example
Transmute:   i  v  a  t
Utilities:   m  L  S
#+end_example

=Image= (=i=) -- 6 columns, 5 rows:

#+begin_example
 Image: Basic           Meta              Enhance          Colour        Organise            Share
       c Convert      d From PDF        z Crush (640px)  b Brighten    o Organise          e Email
       C Crop         i Info            s Scale (1920px) a Auto Colour w WhatsApp Fix      g To Capture
       S Split        x Show Backup ... u Upscale (GAN)  O Orientation F Update from CD
       m Montage      X Set Xattr       r Rotate Right                 U Update to CD
       p To PDF                         l Rotate Left
#+end_example

=Video= (=v=) -- 5 columns, 5 rows:

#+begin_example
 Video: Convert          Trim                 Effects            Sequence          Audio
       c Convert        C Rotate CCW        g To GIF           j Concat          a Extract Audio
       s Shrink         r Reverse           + Speed Up         2 Double          x Remove Audio
       e Rescale        t Top/Tail (Trim)   - Slow Down        E Extract Frames  m Replace Audio
       R Rotate Right   k Cut               f Remove Flips     I From Frames     B Background Music
       L Rotate Left                        F Set FPS
#+end_example

=Audio= (=a=) -- 1 column, 5 rows:

#+begin_example
 Audio: c Convert
       n Normalise
       t Trim Silence
       i Info
       I Video Info
#+end_example

=Tags= (=t=) -- 2 columns, 4 rows:

#+begin_example
 Tags:          Clean:
     t Tag (Interactive)  d Retag by Date
     k Tag from List      c Clear Tags
     r Add Tags & Rename  i Tag Info
     R Rename from Tags
#+end_example

Press =q= or =C-g= at any sub-transient to return to the main menu.

** Monitoring Progress
- Check the top of your windows for the progress bar: =[Transmuting: 5/10 (50%)]=
- View the full command output and history in the =*transmute-log*= buffer (press =L= in the menu).
- To cancel all active tasks, press =S= in the menu.

project's main git repository.

* Versions

** <<2026-05-13 Tue>> *0.4.0*

Display buffer integration -- Transmute commands and menu now work from the full-size image display buffer.

*** Features

- *Display Buffer Integration* -- `transmute-menu` (=C-c M=) is now bound and functional inside the =*image-dired-display-image*= buffer. The file shown in the display buffer is automatically detected and used as the target for all transmute operations.
- *Multi-Backend File Detection* -- New helper `transmute--image-dired-display-file` resolves the displayed file from =buffer-file-name= (full-quality mode) or falls back to the thumbnail buffer's text properties (fast/scaled mode).
- *Hook-Based Keybinding* -- =C-c M= is injected into display buffers via =special-mode-hook= and =image-dired-image-mode-hook=, covering both the fast/scaled and full-quality display paths.
- *Updated Setup* -- `transmute-setup-thumbnail-keys` now also binds =C-c M= in =image-dired-display-image-mode-map=.

** <<2026-05-04 Mon>> *0.3.0*

Major feature expansion migrating legacy media processing bash scripts to pure Elisp.

*** Features

- *Image Migrations* -- Added `transmute-picture-montage`, `transmute-picture-organise`, `transmute-picture-fix-whatsapp`, `transmute-picture-email`, `transmute-picture-from-pdf`, `transmute-picture-orientation-reset`, `transmute-picture-split`, `transmute-picture-to-capture`, and `transmute-picture-update-to-create-date`.
- *Video Migrations* -- Added `transmute-video-concat`, `transmute-video-double`, `transmute-video-extract-frames`, `transmute-video-set-fps`, `transmute-video-from-frames`, `transmute-video-remove-flips`, `transmute-video-replace-audio`, `transmute-video-background-music`, `transmute-video-rescale`, and `transmute-video-rotate-ccw`.
- *Menu Reorganization* -- Refactored the transient menu into a structured multi-column layout for better navigation of the expanded feature set.
- *Sidecar Support* -- Improved handling of companion files (e.g., =.xmp=) during organization and renaming tasks.

** <<22026-05-02 Sat>> *0.2.0*

Significant architectural update to support asynchronous non-blocking media processing.

*** Features

- *Asynchronous Execution* -- All media conversion tasks now run in the background, preventing Emacs from hanging during large batch operations.
- *Header-Line Progress* -- Real-time progress indicator in the window header line (e.g., `[Transmuting: 5/10 (50%)]`) which appears only when tasks are active.
- *Centralized Logging* -- All conversion output, start/end status, and failure details are consolidated in a dedicated =*transmute-log*= buffer.
- *Abort Support* -- Ability to immediately stop all active conversions via `transmute-stop-conversions` (bound to =S= in the menu).
- *Improved Process Management* -- Automatic cleanup of temporary files and individual process buffers after task completion.

** <<22026-04-30 Thu> *0.1.0*

Initial release of Transmute.

*** Features

- *Core Media Processing* -- Native Elisp wrappers for =magick=, =ffmpeg=, =exiftool=, =sox=, and =tesseract= for batch processing of images, video, and audio.
- *Transient Menu* -- Keyboard-driven interface for quickly triggering media operations from Dired.
- *Dired Integration* -- Support for processing marked files or the file at point in Dired buffers.
- *Filename Parsing* -- Intelligent parsing of filenames following the `YYYYMMDDHHMMSS--description__tag1@tag2.ext` pattern.
- *Metadata Preservation* -- Automated metadata copying and modification time restoration using =exiftool= and =set-file-times=.
- *Batch Processing* -- Implementation of `transmute-do-batch` macro for consistent iteration over target files.
- *Image Utilities* -- Support for resizing (crush/scale), rotation, colors correction, GAN upscaling, OCR text extraction, and PDF conversion.
- *Video Utilities* -- Support for conversion to MP4, shrinking, extraction of audio, trimming (top/tail), cutting, reversing, and GIF conversion.
- *Audio Utilities* -- Support for MP3 conversion, volume normalization, and silence trimming.
- *Tagging System* -- Interactive tagging of media files using =exiftool= with hierarchical support.
- *Date-based Renaming* -- Ability to rename files based on EXIF creation dates.
